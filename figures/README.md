# figures / 図版

## English

This directory contains figures used in **Conceptual Physics Visualization**.

Figures are organized by their role in the textbook.

## 日本語

このフォルダには **物理学可視化** で使用する図版を収録します。

図版は教材内での役割ごとに分類しています。

---

## Directory Structure / フォルダ構成

| Folder | English | 日本語 |
|---|---|---|
| pt | Part Title Figures | 部タイトル図 |
| ch | Chapter Header Figures | 章タイトル図 |
| ss | Super Summary Figures | 重要統合図 |
| s | Core Figures | 章の核となる図 |
| a | Supporting Figures | 理解補助図 |
| b | Optional Figures | 追加・発展図 |
| c | Inverse Transform Figures | 逆変換シリーズ |

---

## Naming Convention / 命名規則

### English

Figures are named using a simple ID-based convention.

Examples:

- ch-01.png
- pt-01.png
- ss-01.png
- s-01.png
- a-01.png
- b-01.png
- c-01.png

The figure ID corresponds to the figure number used in the textbook.

---

### 日本語

図版は図番号に対応したシンプルな命名規則を採用しています。

例：

- ch-01.png
- pt-01.png
- ss-01.png
- s-01.png
- a-01.png
- b-01.png
- c-01.png

図番号は教材本文中の図番号と対応しています。

---

## Note / 注意

## English

This directory is for exported figure images used in the textbook and supplementary slides.

Editable source files such as PowerPoint, SVG, or project files should be managed separately if needed.

## 日本語

このフォルダは、教材本文や付録スライドに使用する書き出し済み図版を置くための場所です。

PowerPoint、SVG、編集用プロジェクトファイルなどの元データは、必要になった場合に別途管理します。
