# 物理学可視化

Physics Visualization

## 日本語

本リポジトリは、大学教科書を代替するものではありません。

物理学を「変換」と「構造」の観点から俯瞰し、
フーリエ変換、畳み込み、FFT、波動方程式、電磁気学、量子力学のつながりを可視化するための資料です。

目的は、公式を網羅的に解説することではなく、
一見別々に見える概念がどのような共通構造で結び付いているのかを理解することです。

---

## English

This repository provides a conceptual visualization of physics.

It is not intended to replace university textbooks.
Rather than covering every topic in detail, it focuses on the structural connections between:

* Fourier Transform
* Convolution
* FFT
* Wave Theory
* Electromagnetism
* Quantum Mechanics

These topics are connected through the common concept of **transformation**.

The materials are currently available in Japanese.
