# 物理学可視化

**Conceptual Physics Visualization**

---

## English

Conceptual Physics Visualization is an educational resource that presents physics from the perspectives of **transformation**, **waves**, and **structure**.

This repository is **not intended to replace university textbooks**. Instead of explaining every formula in detail, it focuses on helping learners understand how seemingly independent topics are connected through common mathematical ideas.

The learning path is organized into three parts:

* **Part I – Transformations**

  * Fourier Transform
  * Convolution
  * FFT
  * Wavelet Transform
  * Laplace Transform

* **Part II – Waves**

  * Wave Equation
  * Maxwell's Equations

* **Part III – Quantum Physics**

  * Schrödinger Equation
  * Wave Mechanics = Matrix Mechanics

The final chapter integrates these topics into a unified conceptual view of modern physics.

The materials are currently available in Japanese.

## 日本語

「物理学可視化」は、**変換・波・構造**という視点から物理学全体を俯瞰することを目的とした教育教材です。

本リポジトリは、大学教科書を代替するものではありません。

公式を網羅的に解説することよりも、一見すると独立して見える概念が、どのような共通構造で結び付いているのかを理解できるよう構成しています。

教材は次の3部構成です。

### 第1部　変換の世界

* フーリエ変換
* 畳み込み
* FFT
* ウェーブレット変換
* ラプラス変換

### 第2部　波の世界

* 波動方程式
* マクスウェル方程式

### 第3部　量子の世界

* シュレーディンガー方程式
* 波動力学＝行列力学

最後に「統一の物理学」として、これらを一つの流れとして整理します。

本教材では、「変換」という視点を軸に、数学・波動・量子力学がどのようにつながっているのかを可視化することを目指しています。
